How To Use!

Tampilan Gambar ,Pilihan Search dan Upload

Dalam segmen ini, pengguna dapat mengunggah gambar dalam bentuk file atau folder sebagai dataset pencarian dengan dua parameter. Setelah unggah, pengguna dapat melihat tampilan gambar yang siap diproses. Terdapat dua tombol utama untuk pencarian Content-Based Image Retrieval (CBIR) berdasarkan tekstur dan warna. Selain itu, terdapat fitur Delete Database yang terdiri dari dua tombol, satu untuk menghapus seluruh dataset dan yang lainnya untuk menghapus dataset spesifik hasil scraping.

Interaksi dengan image

pengguna dapat meng-hover cursor mouse ke image-image yang ada. Pengguna akan diberi 2 pilihan button yaitu ‘Select’ dan ‘Hapus’. Sesuai dengan namanya, Select akan menyeleksi gambar tersebut dan dijadikan sebagai query gambar untuk CBIR. Button Hapus berfungsi untuk menghapus gambar yang pengguna pilih dari dataset. Tiap page, gambar diberi maksimal sebanyak 12 buah, dan dapat berganti-ganti page dari pagination di bawah segmen ini. Dengan menambahkan fungsionalitas Select dan Hapus pada gambar-gambar yang ditampilkan, pengguna dapat dengan mudah melakukan aksi yang diinginkan tanpa harus meninggalkan halaman.
